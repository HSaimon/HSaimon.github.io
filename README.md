# 🚀 Portfólio Pessoal — Hítalon Saimon

Este repositório contém o código-fonte do meu portfólio profissional, desenvolvido para centralizar a minha trajetória como **Desenvolvedor Full Stack**. O projeto apresenta de forma dinâmica as minhas competências técnicas, experiências profissionais, formações académicas e os principais projetos integrados com a API do GitHub.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído com foco em performance e design responsivo, utilizando uma arquitetura modular sem a dependência de frameworks pesados.

* **Frontend**: HTML5, CSS3 (com variáveis de sistema para temas) e JavaScript Moderno (ES6+).
* **Animações e UI**: Utilização da **Intersection Observer API** para efeitos de "Scroll Reveal" e CSS Grid/Flexbox para layouts adaptáveis.
* **Integrações**: Consumo da **API do GitHub** para listagem e filtragem dinâmica de repositórios em tempo real.
* **Recursos Visuais**: Tipografia via Google Fonts (Syne e DM Sans) e ícones através das bibliotecas Devicon e Iconify.

## 📁 Estrutura do Projeto

A organização do código segue princípios de modularidade para facilitar a manutenção:

* `assets/`: Documentos (como o currículo em PDF), imagens e ícones de marcas.
* `css/`: Ficheiros de estilização divididos por secções (hero, about, projects, skills, etc.).
* `js/`: Lógica modularizada, incluindo a gestão de temas, cronogramas e integração com o GitHub.
* `index.html`: Ponto de entrada principal do portfólio.

## ✨ Funcionalidades Principais

* **Alternância de Temas (Dark/Light)**: Sistema com persistência de preferência no `localStorage`.
* **Gestão Dinâmica de Projetos**: Script personalizado que aplica uma pontuação para destacar projetos principais no topo da listagem.
* **Timeline Interativa**: Secção de "Sobre" com abas para alternar rapidamente entre Formação, Experiência e Certificações.
* **Órbitas de Tecnologias**: Animação visual no Hero que destaca as stacks principais em movimento orbital.

Esta vitrine demonstra a minha capacidade de transformar requisitos técnicos complexos em soluções intuitivas, elegantes e centradas na experiência do utilizador, refletindo a minha evolução contínua em Engenharia de Software.
